Case study 01 · meta

Using Claude to build this workshop

The most honest demo of a research assistant is to show it doing real work — so the workshop's own preparation is the case study. Below is the actual conversation that planned it: each turn as what I asked → what the agent did → the tools it used → the outcome. Summaries, not raw transcripts.

1

Start the workshop plan

I asked
Raul invited me to give a workshop on AI tools for research. Let's build it from the tools I use — capture the concepts to teach and the tools to showcase.
The agent did
Recognized it as a vault note, ran a dedup check, and filed an idea note with a core insight, the concepts to teach, and a tool list seeded from my real stack — then committed it to Git.
Skill: intelligent-notesGrepWritegit
Outcome
ideas/workshop-ai-tools-for-research.md — the living plan.
2

Track it, and remind me

I asked (voice note)
Add this to my planning repo, aim for July, and nudge me every week or two in the daily briefing.
The agent did
Transcribed the voice note, created a planning project (July deadline), then wrote a new feature into my live morning-briefing script — a recurring "nudge" field — and dry-ran it to confirm before relying on it.
WhisperEditBash: --dry-rungitmemory
Outcome
A tracked project and a biweekly reminder that auto-stops when I mark it done. The agent extended its own automation on request — and tested it before shipping.
3

Make the process the showcase

I asked (voice note)
Let's record our interactions as an example — specifically, a case study of how I'm using you to build this workshop.
The agent did
Made it the centerpiece of the plan and started this annotated log — summarizing, not dumping — with a note to scrub anything sensitive before showing it publicly.
WhisperWritegit
Outcome
This case study — self-referential proof the tools work.
4

Choose the delivery format

I asked
What about an HTML presentation in the browser, with links to the case studies — use the frontend skill to make it look professional?
The agent did
Found the frontend-design skill, committed to a refined technical-editorial aesthetic, and built this very page — the hub and this case study — from the vault notes.
Skill: frontend-designWrite (HTML/CSS)git
Outcome
The page you're reading. The artifact is the argument.
